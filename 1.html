Raspberry Pi - это первое поколение одноплатных миникомпьютеров, разработанных Raspberry Pi Foundation. Основой компьютера служит SoC (система на чипе) Broadcom BCM2835 с процессором ARM1176JZF-S (частота 700 МГц) и видео ускорителем VideoCore IV, поддерживающим Full HD-разрешение. Корневая файловая система, образ ядра и пользовательские файлы размещаются на карте памяти или SDIO. Raspberry Pi работает в основном на операционных системах, основанных на Linux ядре. Производителем поддерживается и обновляется официальный Debian подобный дистрибутив Raspbian. Для облегчения процесса установки операционных систем существует инструмент NOOBS.

Полные технические спецификации:
  • SoC: Broadcom BCM2835
  • CPU: 700 MHz ARM11 ARM1176JZF-S core
  • GPU: Broadcom VideoCore IV, OpenGL ES 2.0,OpenVG 1080p H.264, 250 MHz
  • Оперативная память SDRAM:
    • Model A: 256MB
    • Model B: 256MB (до 15.10.2012), 512MB (с 15.10.2012)
    • Model A+: 256MB
    • Model B+: 512MB
  • USB порты:
    • Model A: 1 USB 2.0
    • Model B: 2 USB 2.0
    • Model A+: 1 USB 2.0
    • Model B+: 4 USB 2.0
  • Видеовыходы:
    • Model A: Композитный видеовыход RCA, HDMI
    • Model B: Композитный видеовыход RCA, HDMI
    • Model A+: HDMI, Композитный видеовыход, совмещённый с 3.5мм аудиовыходом
    • Model B+: HDMI, Композитный видеовыход, совмещённый с 3.5мм аудиовыходом
  • Аудиовыходы: 3.5mm jack, HDMI
  • Аудиовходы: Отсутствуют, но поддерживаются внешние USB звуковые карты с аудиовходом
  • Поддержка карт памяти:
    • Model A: SD, MMC, SDIO
    • Model B: SD, MMC, SDIO
    • Model A+: SD, MMC, SDIO
    • Model B+: MicroSD
  • Сетевые интерфейсы:
    • Model A: Отсутствует
    • Model B: 10/100 Ethernet RJ45
    • Model A+: Отсутствует
    • Model B+: 10/100 Ethernet RJ45
  • Низкоуровневая перефирия:
    • Model A: 26 выводной штырьковый GPIO разъём
    • Model B: 26 выводной штырьковый GPIO разъём
    • Model A+: 40 выводной штырьковый GPIO разъём
    • Model B+: 40 выводной штырьковый GPIO разъём
  • Часы реального времени (RTC): Встроенные RTC отсутсвует.
  • Энергопотребление:
    • Model A: ~300мА, (~1.5 Вт)
    • Model B: ~700мА, (~3.5 Вт)
    • Model A+: ~600мА, (~3.0 Вт)
    • Model B+: ~650мА, (~3.0 Вт)
  • Питание: 5V (DC) через MicroUSB разъём или GPIO
  • Размеры и вес:
    • Model A: 85.0 x 56.0 мм x 15мм, 31г
    • Model B: 85.0 x 56.0 мм x 17мм, 40г
    • Model A+: 65.0 x 56.0 мм x 12мм, 23г
    • Model B+: 85.0 x 56.0 мм x 17мм, 40г
