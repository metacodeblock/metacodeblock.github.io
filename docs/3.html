Raspberry Pi 3 Model B является наиболее актуальной и востребованной на сегодняшний день моделью. Обновлённый одноплатник предложил в разы улучшенное аппаратное обеспечение, сохранив неизменной стоимость для розничного покупателя. Самым значительным обновлением в Raspberry Pi 3 стал 64-битный процессор ARM Cortex-A53 с номинальной частотой 1,2 ГГц. По данным производителя, новая версия должна работать до 10 раз быстрее чипа, установленного в первой версии Raspberry Pi и в среднем на 50–60% быстрее процессора Raspberry Pi 2.

Полные технические спецификации:
  • SoC: Broadcom BCM2837
  • CPU: 1.2GHz 64-bit ARMv8 Cortex-A53 (4 ядра)
  • GPU: Broadcom VideoCore IV, OpenGL ES 2.0,OpenVG 1080p60 H.264, 400 MHz
  • Оперативная память SDRAM: 1024 MB
  • USB порты: 4 USB 2.0
  • Видеовыходы: HDMI, Композитный видеовыход, совмещённый с 3.5мм аудиовыходом
  • Аудиовыходы: 3.5mm jack, HDMI
  • Аудиовходы: Отсутствуют, но поддерживаются внешние USB звуковые карты с аудиовходом
  • Поддержка карт памяти: MicroSD
  • Сетевые интерфейсы: 10/100 Ethernet RJ45, BCM43438 Wi-Fi 802.11n Wi-Fi и Bluetooth 4.1
  • Низкоуровневая перефирия: 40 выводной штырьковый GPIO разъём
  • Часы реального времени (RTC): Встроенные RTC отсутсвует.
  • Энергопотребление: ~800мА (4.0 Вт)
  • Питание: 5V (DC), 2 через MicroUSB разъём или GPIO
  • Размеры и вес: 85.6 x 56.5 мм x 17мм, 45г
