Raspberry Pi Zero - это компьютер за 5$, выпущен в 2015 году. По характеристикам Pi Zero ближе к исходному варианту Raspberry Pi, на той же SoC BCM2835 с частотой 1 ГГц. В Pi Zero есть слот для карт MicroSD, но без привычного механизма защелкивания. 40-контактный GPIO лишился штырьков, а из портов по периметру платы остались только micro-USB для питания и периферийных устройств и mini-HDMI для аудио/видео. В феврале 2017 года выпущена обновленная версия Pi Zero W, в которая добавлен чип беспроводной связи Cypress CYW43438 с поддержкой Wi-Fi и Bluetooth, а антенной является часть макетной платы.

Полные технические спецификации:
  • SoC: Broadcom BCM2835
  • CPU: 1000MHz ARM1176JZ-F с пониженным энергопотреблением
  • GPU: Broadcom VideoCore IV, OpenGL ES 2.0,OpenVG 1080p H.264, 250 MHz
  • Оперативная память SDRAM: 512 MB
  • USB порты: 1 Micro USB OTG
  • Видеовыходы: mini-HDMI (композитный видеовыход не припаян, предусмотрены 2 отверстия для его монтажа)
  • Аудиовыходы: HDMI
  • Аудиовходы: Отсутствуют, но поддерживаются внешние USB звуковые карты с аудиовходом
  • Поддержка карт памяти: MicroSD
  • Сетевые интерфейсы:
    • Pi Zero: Отсутствует
    • Pi Zero W: 802.11n Wi-Fi и Bluetooth 4.1 (Bluetooth Classic и LE)
  • Низкоуровневая перефирия: 40 выводной GPIO на плате (без штырьков)
  • Часы реального времени (RTC): Встроенные RTC отсутсвует.
  • Энергопотребление: ~160мА
  • Питание: 5V (DC) через MicroUSB разъём или GPIO
  • Размеры и вес: 65.0 x 30.0 мм x 5мм, 9г
